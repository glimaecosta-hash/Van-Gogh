FrontEnd

Front-end é a parte de um website ou aplicação com a qual o usuário interage diretamente. É tudo aquilo que você vê na tela do seu navegador, tablet ou celular. Pense no Front-end como a fachada de uma loja: as vitrines, a decoração, as cores e a disposição dos produtos. O objetivo do Front-end é garantir uma boa Experiência do Usuário (UX), ou seja, que o site seja bonito, fácil de usar, rápido e funcione corretamente em diferentes dispositivos.


Os três Pilares do Front-end

  1. HTML (HyperText Markup Language): É a estrutura ou o esqueleto da página. Define o conteúdo e a sua organização (onde estão o texto, as imagens, os links, os botões, etc.).
  2. CSS (Cascading Style Sheets): É a estilização ou a "maquiagem". Define a aparência da estrutura, controlando cores, fontes, layouts, espaçamento e responsividade (como o site se ajusta a diferentes tamanhos de tela).
  3. JavaScript (JS): É a interação ou o "comportamento". Adiciona dinamismo e funcionalidade, permitindo que o site faça coisas, como validar formulários, carregar novos dados sem recarregar a página, criar animações e gerenciar o slideshow que você estava desenvolvendo.

Front-end vs. Back-end

É importante entender que o Front-end é a ponta visível de uma aplicação. A outra metade é o Back-end. O Front-end é a parte visual e interativa, enquanto o Back-end é o motor por trás, responsável pelo processamento e armazenamento de informações.
